Registration Form – All Input Types

What is a Registration Form?

A registration form collects user information such as name, email, password, gender, skills, address, etc., and sends it to a server for processing.

Understanding action & method

action: Specifies WHERE the form data should be sent.

Example: action="/register" → sends form data to server route "/register"

method: Specifies HOW data is sent to the server.

  • GET
    • Appends data in URL
    • Used for search bars, filters
    • Not secure
  • POST
    • Sends data securely in background
    • Used for login, registration, payments
    • More secure & recommended
