HTML – Introduction & Overview

What is HTML?

HTML (HyperText Markup Language) is the standard language used to create and structure webpages. It uses elements (tags) to define headings, paragraphs, images, links, tables, forms, and overall page layout.

History of HTML

  • 1991 – HTML was created by Tim Berners-Lee.
  • 1993 – HTML 1.0 released as the first official version.
  • 1995 – HTML 2.0 standardized by IETF.
  • 1997–1999 – HTML 3.2 and HTML 4.01 released by W3C.
  • 2014 – HTML5 became the official recommendation.
  • Till today, HTML5 continues to evolve with new APIs and browser features.

Versions of HTML

  • HTML 1.0 (1993)
  • HTML 2.0 (1995)
  • HTML 3.2 (1997)
  • HTML 4.01 (1999)
  • XHTML 1.0 (2000)
  • HTML5 (2014)
  • HTML Living Standard (ongoing updates)

Features of HTML5

  • Supports audio and video without plugins (<audio>, <video>)
  • Semantic tags like <header>, <footer>, <article>
  • Offline support with localStorage & sessionStorage
  • Canvas API for graphics and animations
  • Improved forms with new input types
  • Geolocation API
  • Responsive design support with viewport and media features

Limitations of HTML

  • Cannot create dynamic content without JavaScript
  • No built-in logic or programming capabilities
  • Depends on CSS for styling and layout
  • Browser compatibility issues sometimes occur
  • Limited security features
