HTML5 Features – Concept Understanding

1. New Semantic Elements

HTML5 introduced elements that describe the meaning of content, improving SEO and accessibility.

  • <header>
  • <footer>
  • <nav>
  • <section>
  • <article>
  • <aside>

2. Multimedia Support

HTML5 removed the dependency on external plugins for media playback.

  • <audio> – Plays audio files
  • <video> – Plays videos

3. Graphics & Animation

  • <canvas> – Used for drawing 2D graphics
  • SVG – Scalable Vector Graphics for icons & illustrations

4. Form Enhancements

HTML5 added new input types and attributes for better UX and validation.

  • Email, Date, Time, Number, URL, Search, Range, Color
  • Attributes like required, minlength, pattern, autocomplete

5. Local Storage Mechanisms

  • localStorage – Stores data permanently
  • sessionStorage – Stores data until tab is closed

6. Geolocation API

Allows websites to access user's location (with permission).

7. Offline & Caching Support

  • Service Workers
  • Manifest files

8. Web Workers

Allows background processing without freezing the UI.

9. Drag & Drop API

Allows elements to be dragged and dropped easily.
