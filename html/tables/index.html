HTML Tables – Syntax, Example & Real-Time Use Cases

What Are HTML Tables?

HTML Tables are used to display data in rows and columns, forming a structured grid format. They are useful for showing information that naturally fits into a tabular layout.

Main Table Tags:

  • <table> – Main table wrapper
  • <tr> – Table Row
  • <th> – Header Cell (bold + centered by default)
  • <td> – Data Cell
  • <thead> – Group of header rows
  • <tbody> – Group of body rows
  • <tfoot> – Footer rows
  • <caption> – Table title

HTML Table Syntax

Basic Syntax:

<table>
  <tr>
    <th>Header</th>
    <th>Header</th>
  </tr>
  <tr>
    <td>Data</td>
    <td>Data</td>
  </tr>
</table>

Real-Time Use Cases of Tables

HTML tables are commonly used in many web applications and dashboards. Some examples:

  • eCommerce Admin Panels: Product lists, order history, customer details
  • School/College Portals: Timetables, marksheets, attendance records
  • Finance Dashboards: Transaction history, profit–loss sheets
  • HR Systems: Employee records, payroll data
  • Hospital Management: Patient records, doctor schedules
  • Travel Websites: Flight schedules, booking details
  • Restaurant POS Systems: Menu pricing, daily sales
