HTML Tags – Definitions & Usages

Typography Tags

Typography tags define text formatting, headings, emphasis, and inline styling.

  • <h1> – <h6> : Defines headings from highest to lowest priority Usage: <h1>Page Title</h1>
  • <p> : Defines a paragraph Usage: <p>This is a paragraph.</p>
  • <br> : Inserts a line break
  • <strong> : Strong importance (bold) Usage: <strong>Important</strong>
  • <em> : Emphasized text (italic) Usage: <em>Note</em>
  • <span> : Inline text container Usage: <span style="color:red">Red Text</span>
  • <mark> : Highlights text Usage: <mark>Highlight</mark>
  • <blockquote> : For long quotations
  • <code> : Displays inline code
  • <pre> : Preformatted text block

Layout & Structure Tags

These tags define webpage layout, structure, and container sections.

  • <div> : Generic block container Usage: <div>Content</div>
  • <section> : Groups related content
  • <article> : Independent content piece (blogs, posts)
  • <header> : Header area of page or section
  • <footer> : Footer area
  • <nav> : Navigation links block
  • <aside> : Sidebar content
  • <main> : Main content region
  • <span> : Inline layout wrapper

List Tags

HTML provides several list formats: ordered, unordered, and descriptive.

  • <ul> : Unordered list (bullets) Usage: <ul><li>Item</li></ul>
  • <ol> : Ordered list (numbers)
  • <li> : A list item
  • <dl> : Description list
  • <dt> : Term
  • <dd> : Description

Media Tags

Used to embed images, audio, video, and graphics.

  • <img> : Embeds an image Usage: <img src="photo.jpg" alt="Image">
  • <audio> : Embeds sound/audio
  • <video> : Embeds a video
  • <source> : Defines media file source
  • <canvas> : Draw graphics with JS
  • <iframe> : Embeds external page/video

Table Tags

Tags used to create tables and tabular data.

  • <table> : Table container
  • <tr> : Table row
  • <th> : Table header cell
  • <td> : Table data cell
  • <thead> : Table header group
  • <tbody> : Body content group
  • <tfoot> : Footer group
  • <caption> : Table title

Form & Input Tags

Form tags allow capturing user data and sending it to servers.

  • <form> : Form container Usage: <form action="" method=""></form>
  • <input> : Multi-purpose input field Types: text, email, password, number, checkbox, radio, etc.
  • <label> : Text label for input
  • <textarea> : Multi-line text box
  • <select> : Dropdown list
  • <option> : Item inside dropdown
  • <button> : Clickable button
  • <fieldset> : Groups form fields
  • <legend> : Label for fieldset

Interactive & Advanced Tags

Used for collapsible content, dialogs, and interactive widgets.

  • <details> : Expandable/collapsible section
  • <summary> : Caption for details
  • <dialog> : Dialog box / popup
  • <progress> : Progress bar
  • <meter> : Gauge/measurement scale

Metadata & Head Tags

These tags define additional information about the document.

  • <title> : Page title shown in browser tab
  • <meta> : Metadata (SEO, charset, author)
  • <link> : Links external resources (CSS)
  • <script> : Embeds JavaScript
  • <style> : Internal CSS
  • <base> : Base URL for links
