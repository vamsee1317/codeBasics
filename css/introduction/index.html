CSS Introduction – Concept Understanding

What is CSS?

CSS (Cascading Style Sheets) is used to control the appearance and layout of HTML elements.

HTML defines the structure, while CSS defines:

  • Colors
  • Fonts
  • Spacing
  • Layouts
  • Animations & responsiveness

Why CSS is Required?

  • Separates content from design
  • Makes websites visually appealing
  • Improves maintainability
  • Enables responsive design
  • Reduces code duplication

What Does "Cascading" Mean?

Cascading means CSS follows a priority order when multiple styles apply to the same element.

The browser decides which rule to apply based on:

  • Style source (inline, internal, external)
  • Specificity
  • Order of declaration

Types of CSS

  • Inline CSS – Applied directly inside HTML tag
  • Internal CSS – Written inside <style> tag
  • External CSS – Written in a separate .css file

CSS Priority Order (Very Important)

When multiple CSS rules target the same element, the priority is:

  1. Inline CSS (Highest priority)
  2. Internal CSS
  3. External CSS
  4. Browser Default Styles

Note: If !important is used, it overrides all normal rules.

Real-Time Example

If a button has:

  • External CSS → color: blue
  • Internal CSS → color: green
  • Inline CSS → color: red

Final color shown: Red (Inline wins)
